# Curso de Banco de Dados - SQL no DevDojo (42h)

Bem-vindo ao repositório sobre o curso de **Banco de Dados - SQL**, realizado na plataforma **DevDojo**. Neste curso, com um total de 42 horas de conteúdo, você aprenderá os conceitos fundamentais e avançados de SQL, abrangendo desde instruções básicas até tópicos mais complexos, como Stored Procedures, Transações e Controle de Fluxo. O curso é voltado para iniciantes e profissionais que desejam aprofundar seus conhecimentos em SQL e bancos de dados relacionais.

---

## Conteúdo do Curso

Ao longo deste curso, você explorará diversos aspectos do SQL e suas aplicações em sistemas de banco de dados. O conteúdo do curso abrange desde o básico até tópicos mais avançados, incluindo modelagem de dados, funções, subconsultas, e operações de agregação, entre outros. Abaixo estão os principais tópicos abordados:

### 1. **Instruções Básicas de SQL**
- **SELECT**: Seleção de dados da tabela.
- **SELECT (TOP, DISTINCT, ORDER BY)**: Técnicas para selecionar dados específicos e ordenar.
- **SELECT (ORDER BY, OFFSET FETCH)**: Controle avançado de resultados com paginação.
- **Operadores Aritméticos**: Utilização de operadores matemáticos para cálculos com dados.
- **Tipos de Dados**: Conhecimento sobre os tipos de dados em SQL e como utilizá-los corretamente.

### 2. **Operadores e Comparações**
- **Operadores de Comparação e Lógicos**: Comparação de valores e condições lógicas, fundamentais para filtrar e segmentar dados.
- **Information Schema**: Como acessar a estrutura interna do banco de dados e obter informações sobre as tabelas e colunas.
- **Funções de Agregação**: Contagem, soma, média, máximo, mínimo, etc., para sumarizar dados.

### 3. **Joins e Subconsultas**
- **INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN**: Como unir tabelas de maneira eficiente, combinando dados de diferentes fontes.
- **UNION**: Combinação de resultados de múltiplas consultas, permitindo que você agregue conjuntos de dados distintos.
- **Subconsultas (Subqueries)**: Consultas dentro de outras consultas, permitindo realizar operações mais complexas e específicas.

### 4. **Funções e Manipulação de Dados**
- **Funções de Data e Hora**: Funções como DATEPART, DATENAME, e DATEDIFF, para realizar cálculos e manipulação de dados temporais.
- **Funções de Texto**: Como manipular strings com funções como SUBSTRING, CHARINDEX, REPLACE, essencial para a limpeza e análise de dados textuais.
- **Trabalhando com NULL**: Funções como ISNULL e COALESCE para lidar com valores nulos, um conceito importante para garantir a integridade dos dados.

### 5. **Comandos de Manipulação de Dados**
- **INSERT, UPDATE, DELETE**: Como inserir, atualizar e excluir dados de tabelas de forma eficiente e segura.
- **ALTER TABLE**: Como alterar a estrutura das tabelas existentes, adicionar ou remover colunas.

### 6. **Modelagem de Dados e Normalização**
- **Modelo Lógico e Físico**: Conceitos de modelagem de dados e como criar tabelas adequadas para a eficiência do banco de dados.
- **Normalização**: Técnicas para normalizar dados e evitar redundâncias, como a 1ª, 2ª e 3ª formas normais.

### 7. **Funções Avançadas de SQL**
- **Funções de Classificação (ROW_NUMBER, RANK, DENSE_RANK)**: Funções de janela para classificar resultados com base em critérios específicos, essenciais para análises de dados.
- **Funções Analíticas (LAG, LEAD, FIRST_VALUE, LAST_VALUE)**: Análises temporais e comparações entre valores anteriores e subsequentes.

### 8. **Transações e Controle de Fluxo**
- **Transações SQL**: Controle de operações em bancos de dados com comandos como `BEGIN TRANSACTION`, `COMMIT`, e `ROLLBACK`. Isso é fundamental para garantir a integridade dos dados durante operações complexas.
- **Controle de Fluxo**: Comandos como `IF ELSE`, `WHILE`, `GOTO` e `TRY CATCH` para garantir que o SQL execute ações de acordo com condições e fluxos lógicos.

### 9. **Stored Procedures e Funções**
- **Stored Procedures**: Criação e execução de procedimentos armazenados, que encapsulam lógicas complexas e podem ser reutilizados.
- **Funções SQL**: Criação de funções personalizadas para encapsular lógicas que podem ser usadas em várias consultas.

### 10. **Outros Tópicos Avançados**
- **Índices**: Melhoria de performance com a criação de índices, permitindo buscas mais rápidas e eficientes.
- **Views**: Criação de visões para simplificar consultas complexas, tornando o acesso aos dados mais fácil e eficiente.
- **Merge e Output**: Operações para manipulação de dados de maneira eficiente, com comandos que atualizam ou inserem dados com base em uma condição.

## Objetivos do Curso

Este curso foi projetado para proporcionar uma compreensão sólida sobre SQL, cobrindo desde os conceitos básicos até as técnicas avançadas utilizadas em grandes bancos de dados. Após completar o curso, você será capaz de:

- Criar e manipular bancos de dados, tabelas e dados de forma eficiente.
- Realizar consultas avançadas utilizando **JOINs**, **subconsultas** e funções analíticas.
- Lidar com dados **nulos**, manipular **strings** e realizar **conversões de tipos de dados**.
- Aplicar **controle de fluxo** e **transações** em SQL para garantir integridade e precisão nas operações.
- Criar **procedimentos armazenados (Stored Procedures)** e funções personalizadas para encapsular lógicas complexas.
